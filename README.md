# children-help-server
基于SpringBoot+Vue特殊儿童家长互助平台，系统包含管理员、专家、用户角色，功能如下： 管理员：首页统计；用户管理；专家管理；互助论坛；资源管理（资源分类、资源管理）；活动管理（活动分类、活动管理、活动报名）；在线咨询；预约管理；评价管理；平台留言；系统公告；个人信息；密码修改。 专家：注册；登录；资源分类；资源管理；预约管理；评价管理；公告信息；修改个人资料；修改密码。 用户：注册、登录系统；首页；互助论坛（论坛版块、帖子列表、详情、点赞、评论）；资源库（教育资源、治疗资源、政策法规）；活动支持；专家咨询（详情、咨询、预约、评价）；平台留言；公告信息。用户还具有个人中心功能，功能包括：我的帖子、帖子评论、资源收藏、活动报名、专家咨询、专家预约、专家评价、平台留言。
