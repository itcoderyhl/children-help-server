## 【原创】springboot+vue前后端分离的特殊儿童家长互助平台

## 系统介绍

基于SpringBoot+Vue前后端分离的特殊儿童家长互助平台，系统包含管理员、专家、用户角色，功能如下：
- 管理员：首页统计（资源统计、活动统计、专家预约统计）；用户管理；专家管理；互助论坛（板块管理、帖子管理、帖子评论）；资源管理（资源分类、资源管理）；活动管理（活动分类、活动管理、活动报名）；在线咨询；预约管理；评价管理；平台留言；系统公告；个人信息；密码修改。
- 专家：注册；登录；资源分类查看；资源管理发布；预约管理（处理治疗）；评价管理（回复）；公告信息；修改个人资料；修改密码。
- 用户：注册、登录系统；首页；互助论坛（论坛版块、帖子列表、详情、点赞、评论）；资源库（教育资源、治疗资源、政策法规）；活动支持（活动分类、活动详情、报名）；专家咨询（详情、咨询、预约、评价）；平台留言；公告信息。用户还具有个人中心功能，功能包括：我的帖子、帖子评论、资源收藏、活动报名、专家咨询、专家预约、专家评价、平台留言、个人资料、修改密码。
系统界面美观大方，功能丰富，使用了SpringBoot、Vue、ElementUI、Echarts等技术栈，适合作为毕业设计、课程设计，有需要可提供指导。

项目由本人原创开发，经过多次测试，项目运行无问题，请放心使用，作者QQ：1344127185。

## 使用技术

服务器端技术栈：

- Springboot
- SpringMvc
- Mybatis
- MySQL

前端技术栈：

- vue
- elementUI
- axios
- echarts

## 环境介绍

- IDEA集成环境
- JDK1.8
- MySQL5.6+

可提供所有运行软件、远程安装、技术指导。
